Tag Selector for jQuery

Allows selecting multiple "Tags" by using jQuery's Autocomplete plugin with an interface similar to the one used by Facebook for sending a message to friends.

Usage:

In your page you need to include jQuery, the Tag Selector JavaScript file and the Tag Selector CSS file. In the following sample you can see how it looks: 

<link rel="stylesheet" href="jquery-tagselector.css">
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.4.2/jquery.min.js"></script>
<script src="jquery-tagselector.js"></script>
<div class=tagSelector>
	<span class=tag>
		<input name=tags type=hidden value=42>
		jQuery
	</span>
</div>
<script>
$(function() {

	// Display a message on load.
	$('#tagSelector').tagSelector();

});
</script>

Methods

jQuery.flashMsg(msg)

Displays the specified message. It's equivalent to calling jQuery.flashMsg('show', msg).

jQuery.flashMsg('setup', options)

Initializes the message display. If you don't call it, it will be initialized using the default settings. You can specify any of the following options:

Parameter Type Default Description
id string "flashMsg" Id of the message display.
displayTime int 5000 Time to display the message in milliseconds. By default it's 5 seconds.
hideByUser bool true Hides the message before displayTime has elapsed if it detects user activity (mouse movement or a keypress).
opacity double 0.8 Opacity of the message display. 0-1
jQuery.flashMsg('show', msg)

Displays the specified message.

jQuery.flashMsg('hide')

Hides the message display.
